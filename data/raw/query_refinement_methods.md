Hypothetical Document Embedding (HyDE) leverages an LLM to generate a synthetic response to a user query. The synthetic document is then embedded and used to query a vector index, bypassing lexical mismatch bottlenecks. Query rewriting generates multiple alternative query formulations, broadening term expansion for keyword retrieval. BM25 is a term-matching algorithm that performs poorly on synthetic narratives due to lexical differences between synthetic and actual texts.
